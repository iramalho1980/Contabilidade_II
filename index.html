Integração Contábil dos Documentos Fiscais

▼

Automática

  • Ocorre no momento que o arquivo XML é importado
  • Exceto Sat CF-e que deve ser integrado de outra forma

Acumuladores

  • Configuração para integração das notas fiscais
  • Acesse: //Arquivos/Acumuladores aba //Contabilidade/Notas (informar as contas contábeis)
  • Já vem pré configurado (Plano padrão)

Integração Contábil dos Impostos Apurados

▼

Cadastro do Imposto

  • Configuração das contas para contabilizar os impostos apurados
  • Acesse: //Arquivos/Impostos aba //Contabilidade/Geral
  • Já vem pré configurado (Plano padrão)

Rotina de Integração

  • Após apuração, subir os impostos para contabilidade
  • Acesse: //Movimentos/Integração Contábil
  • É possível realizar a automação através de rotina automática

Integra Pagamento do Imposto via E-CAC

  • É possível integrar o imposto para o balancete importando diretamente do e-cac
  • Acesse: //Movimentos/Pagamento de impostos via ecac/Escrita Fiscal

Integração Contábil - Folha de Pagamentos

▼

Parâmetros da Empresa

▼
  • Marcar para integrar a folha para Contabilidade
  • Acesse: //Controle/Parâmetros aba //Contabilidade (configurar os campos)
📖 Ver Documentação

Rotina de Integração

▼

Realizada mensalmente após fechamento da folha de pagamentos

Integrar Folha

  • Acesse: //Processos/Integração contábil
  • Informar período
  • Marcar opções necessárias e gerar

Configurar Novas Rubricas

  • Sempre que for criada nova rubrica
  • Já vem configurado para rubricas existentes
  • Acesse: //Arquivo/Contabilidade/Configurar integração
  • Informar as contas contábeis
  • Selecionar a rubrica recém criada

Integra Pagamento da Folha

▼
  • Integrar os pagamentos da folha e encargos para Contabilidade
  • Acesse: //Processos/Pagamentos
  • Selecionar a folha ou encargo
  • Clicar em Pagar
  • Após informar data e valor pago, clicar em Gerar do quadro "Lançamentos Contábeis"
